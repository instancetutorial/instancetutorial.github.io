Overview

The purpose of this tutorial is to discuss the recent advances on instance-level recognition. We will cover in detail the most recent work on object detection, instance segmentation and human pose prediction from a single image. We will show how solid progress on image classification with transferability to instance-level recognition is crucial for better performance. Finally, we will present how instance-level recognition can benefit tasks such as human-object interactions, visual relationships and instance-level reasoning.
